### 第二步：导入 NotebookLM

> 什么是 NotebookLM？
> **NotebookLM** 是由谷歌开发的AI私人知识管理库，可以帮助您快速搜索、分析和理解大量文本资料。

#### 导入步骤：

**① 访问 NotebookLM --> 新建笔记本**

📥 **访问地址：[notebooklm.google.com](https://notebooklm.google.com/)**


**② 上传文件**
- 在新建的笔记本里，点击 **「+ 添加来源」** 按钮
- 选择 **「上传文件」** 选项
- 依次上传所有20个文件
